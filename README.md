# Art of Effective Communication

## Presentation Topic
How to Make Clear and Structured Documents

## Presentation Date
20 April 2026

## Purpose
The purpose of this presentation is to explain practical methods for writing documents that are easy to read, easy to understand, and easy to act on.

## Learning Objectives
Participants should be able to:
- define what makes a document clear and structured
- organize ideas in a logical flow
- write concise and reader-focused sentences
- design document sections that improve navigation
- revise documents using a simple quality checklist

## Agenda
1. Why clarity matters in communication
2. Common problems in poorly structured documents
3. Core principles of clear writing
4. Building strong document structure
5. Practical example: before and after revision
6. Checklist for editing and final review
7. Q and A

## Core Principles Covered
- Audience first: write for the reader's needs and level
- One purpose per document: define the main message early
- Logical hierarchy: use titles, headings, and subheadings consistently
- Concise language: remove unnecessary words and jargon
- Visual clarity: use bullets, spacing, and formatting for scanability
- Actionable endings: state decisions, next steps, or requests clearly

## Recommended Structure for Any Document
1. Title
2. Context or background
3. Objective
4. Key points or analysis
5. Recommendations or decisions
6. Next steps

## Activity (Optional)
*Participants review a short unstructured text and rewrite it using the framework from the session.*

## Final Checklist
Before sharing your document, confirm:
- The purpose is clear in the first paragraph.
- Headings reflect the content of each section.
- Paragraphs are short and focused on one idea.
- Key actions or decisions are easy to find.
- Grammar, tone, and formatting are consistent.
